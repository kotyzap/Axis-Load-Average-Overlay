How it works

Straight from /proc/loadavg to your overlay

No polling agents, no middleware. The app runs on the camera itself and talks directly to CamOverlay's Custom Graphics API.

1. ReadPolls /proc/loadavg on the camera at a configurable interval (default: every 5 seconds).
2. PushSends 1‑minute, 5‑minute, and 15‑minute load averages to a CamOverlay Custom Graphics service.
3. ColorEvery value is color-coded automatically — green under 2.0, orange between 2.0–3.0, red above 3.0 — so severity reads instantly.
4. Deploy anywhereWorks identically on a local network camera or one reachable only through CamStreamer Cloud (device-connect.net).
2.0–3.0 — warning, shown in orange
Above 3.0 — critical, shown in red

Screenshots

See it in action

From the live camera view to the settings dashboard — here's what running Load Average Overlay actually looks like.

A settings UI that shows its own status

Configure the output camera, CamOverlay Custom Graphics fields, cloud routing, and update interval — all from one screen, with live load values and the last push result shown at the top.

  • Real-time 1/5/15-minute load preview while you configure
  • Push status with HTTP response details
  • Field-by-field toggles for exactly what gets sent

Dark theme included

Switch themes with one click — handy when you're configuring cameras from a dim NOC or matching your existing tooling.

Separate field mode

Show 1, 5, and 15-minute load averages as three independent overlay fields, each individually color-coded — ideal when you want the detail broken out at a glance.

Combined field mode

Prefer a single, compact readout instead of three separate boxes? Combined mode merges 1/5/15-minute values into one customizable string, like Load: {load1} {load5} {load15}.

Clean, unobtrusive on-screen readout

Whichever mode you choose, the overlay sits quietly in a corner of the frame — informative without getting in the way of the footage that matters.

Installs like any other micro app

Managed through CamScripter's own micro app manager, alongside your other apps — enable, disable, or update it in a couple of clicks.
